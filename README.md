# Python教育资源信息管理系统（源码+SQL文件）

## 核心功能
- **书籍管理**：对图书资源进行录入、查询、修改和删除。
- **教师管理**：教师信息录入、查询、修改和删除。
- **实验管理**：实验课程信息录入、查询、修改和删除。
- **工资管理**：教师工资信息录入、查询、修改和删除。
- **教室管理**：教室资源信息录入、查询、修改和删除。
- **系统配置**：系统设置和配置管理。
- **物理器材管理**：物理器材信息录入、查询、修改和删除。

## 适用场景
- 教育机构内部管理
- 教研室资料管理
- 学校图书资源管理
- 教师工资统计
- 教室和器材资源分配

## 技术特点
- 采用 Django 框架，支持快速开发。
- MySQL 数据库，保证数据存储的安全性和稳定性。
- HTML、CSS 和 JavaScript，实现前端页面和交互功能。
